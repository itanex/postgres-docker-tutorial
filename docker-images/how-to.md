# How to run the shell scripts

## Run the provided scripts

1. Open a bash terminal
2. Navigate to script folder
3. Run `./create-postgres-docker.sh`
4. Run `./create-pgadmin-docker.sh`

> Note: If you already have containers that have these names, you will just need to modify the names and adjust accordingly for those changes in the rest of this tutorial

## Run the provide script commands

1. Open a terminal window
2. Open a script file
3. Copy the file contents
4. Paste and Execute the file contents
